# 🤖 VK-бот для сообщества «Краски и грунтовки оптом и в розницу»

Бот для автоматизации продаж в сообществе ВКонтакте с приёмом оплаты через ЮКасса.

## Что умеет бот

- 🛒 **Каталог товаров** — 5 позиций грунтовок FITLEX с описаниями и ценами
- 📦 **Корзина** — добавление, удаление товаров, подсчёт итога
- 💳 **Оплата онлайн** — безопасная оплата через ЮКасса с чеком по 54-ФЗ
- ❓ **Консультация** — FAQ по выбору грунтовки, расходу, доставке, оплате
- ℹ️ **О компании** — реквизиты, контакты, адрес
- 🏷 **Распознавание товаров** — когда покупатель нажимает «Написать продавцу» из карточки товара, бот автоматически показывает этот товар с кнопкой «В корзину»

## 🚀 Деплой на Railway (рекомендуется)

Railway — это облачная платформа, где бот будет работать 24/7. Стоимость ~$5/мес.

### Шаг 1: Создать аккаунт на GitHub

1. Зайдите на [github.com](https://github.com) → **Sign up**
2. Подтвердите почту

### Шаг 2: Загрузить код на GitHub

1. Зайдите на [github.com/new](https://github.com/new)
2. Название репозитория: `fitlex-vk-bot`
3. Выберите **Private** (приватный)
4. Нажмите **Create repository**
5. Загрузите файлы (кнопка **uploading an existing file**):
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `runtime.txt`
6. Нажмите **Commit changes**

### Шаг 3: Подключить Railway

1. Зайдите на [railway.app](https://railway.app) → **Login with GitHub**
2. Нажмите **New Project** → **Deploy from GitHub repo**
3. Выберите репозиторий `fitlex-vk-bot`
4. Railway начнёт сборку автоматически

### Шаг 4: Добавить переменные окружения

В Railway откройте проект → вкладка **Variables** → добавьте:

| Переменная | Значение | Где взять |
|------------|----------|-----------|
| `VK_TOKEN` | Токен группы ВК | Управление сообществом → Работа с API → Ключи доступа |
| `YOOKASSA_SHOP_ID` | ID магазина | Личный кабинет ЮКасса → Настройки |
| `YOOKASSA_SECRET_KEY` | Секретный ключ | Личный кабинет ЮКасса → Настройки → Ключи API |

### Шаг 5: Проверить

После добавления переменных Railway автоматически перезапустит бота.
Напишите сообщение в сообщество — бот должен ответить.

## ⚙️ Настройки сообщества ВКонтакте

Убедитесь, что в настройках сообщества включено:

1. **Сообщения сообщества**: Управление → Сообщения → Сообщения сообщества → **Включены**
2. **Long Poll API**: Управление → Работа с API → Long Poll API → **Включено**
3. **Версия API**: `5.199`
4. **Типы событий**: Входящее сообщение ✅

## 📝 Как изменить товары или цены

Откройте файл `bot.py` и найдите раздел `КАТАЛОГ ТОВАРОВ` (примерно строка 75).

Каждый товар выглядит так:
```python
{
    "id": 1,                                          # Уникальный номер
    "name": "Грунтовка FITLEX Elite Edition 5л",      # Название
    "price": 670,                                      # Цена в рублях
    "description": "Описание товара...",               # Описание
},
```

Чтобы **изменить цену** — поменяйте число в `"price"`.
Чтобы **добавить товар** — скопируйте блок и измените `id`, `name`, `price`, `description`.
Чтобы **удалить товар** — удалите весь блок `{...}`.

После изменений сохраните файл на GitHub — Railway автоматически обновит бота.

## 💳 Переключение тест/боевой режим ЮКасса

**Тестовый режим** (без реальных денег):
- В Railway поменяйте `YOOKASSA_SHOP_ID` и `YOOKASSA_SECRET_KEY` на тестовые значения
- Тестовая карта: `1111 1111 1111 1026`, срок — любой будущий, CVV — `000`

**Боевой режим** (реальные платежи):
- Используйте боевые `YOOKASSA_SHOP_ID` и `YOOKASSA_SECRET_KEY`

## 🔧 Решение проблем

| Проблема | Решение |
|----------|---------|
| Бот не отвечает | Проверьте в Railway → Deployments, что статус **Active** |
| «Long Poll not enabled» | Включите Long Poll API в настройках сообщества |
| «Ошибка создания платежа» | Проверьте ключи ЮКасса в переменных Railway |
| Бот отвечает приветствием на кнопки | Перезапустите деплой в Railway (Deployments → Redeploy) |
| Минимальный заказ 2000₽ | Это настройка в коде (bot.py, строка с `total < 2000`) |
